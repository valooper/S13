¿Qué es un framework CSS?

Un framework CSS es una librería que proporciona estilos y componentes predefinidos para acelerar el desarrollo front-end. Ejemplos populares: Bootstrap, Foundation, Bulma, Tailwind CSS.

Ventajas: coherencia visual, componentes reutilizables, sistema de grid y utilidades. Inconvenientes: posible sobrepeso si se usa sin optimizar.

En este ejemplo usamos Foundation para mostrar su grid, tabs, slider (Orbit) y formularios.

Tipos de frameworks CSS

  • Frameworks completos (Bootstrap, Foundation): ofrecen componentes, grid y JS.
  • Frameworks de utilidades (Tailwind): clases pequeñas para construir diseño.
  • Frameworks basados en componentes (Bulma): CSS modular sin JS obligatorio.

La elección depende del proyecto: prototipado rápido, control fino del diseño, rendimiento, y preferencias del equipo.
